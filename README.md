# Electrical Project Manager

This is separate from the Material Delivery Alerts app.

## What it includes

- Job blocks that are quick to start
- Automatic folder shortcuts for Submittals, Drawings, Change Orders, RFIs, Permits, Inspections, Photos, and Closeout
- Drag-and-drop quick access records for files or notes
- Project-only material tracking that does not update the material tracking app
- Tasks and contacts

## Online setup

1. Create a separate Supabase project for this app.
2. Open Supabase SQL Editor and run schema.sql.
3. Copy the Project URL and anon public key into config.js.
4. Upload this folder to GitHub Pages or another static website host.

Drag-and-drop currently records file names and notes. Full file upload/storage can be added next with Supabase Storage.
